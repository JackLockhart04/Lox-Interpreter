# Use

Use Rusts cargo to build to an executable

### Run

    Run `cargo run` to use stdin
    Run `cargo run -- <filename>` to use a file as input

### Build

    Run `cargo build` for debug build
    Run `cargo build --release` for deployment build

### Testing

    Run `cargo test` for normal
    Run `cargo test --test '*'` for easier to read output

# Git

### Add files

    Run `git add .` to add all files

### Commit

    Run `git commit -m "<message>"` to commit to local repo

### Push

    Run `git push origin main`

# Limitations

If statements have to wait until next statement to print out since it has to watch for else
